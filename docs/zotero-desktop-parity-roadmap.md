# Zotero Desktop Parity Roadmap

## Principle

1. **Replicate Zotero Desktop core usage paths first**
2. **Localization (zh-Hans) second**
3. **HarmonyOS-specific capabilities last**

## P0 — Core Workflow (current phase)

- [x] New Item dialog (type picker + title)
- [x] New Collection dialog (named, child collection aware)
- [x] Add by Identifier dialog (DOI/ISBN/PMID/arXiv stub)
- [ ] Import / Export wizard (BibTeX, RIS, CSL JSON)
- [x] Item editing (field-level, draft→Save)
- [x] Note / Attachment child item editing
- [x] Trash / restore
- [x] Quick search (title/creator/date filter)
- [x] Pending sync visibility (dot markers, footer count)

## P1 — Reader / Attachment

- [ ] PDF open (embedded reader or external app)
- [ ] Annotation list in reader
- [ ] Attachment metadata edit (title, contentType, path)
- [ ] Attachment preview/thumbnail
- [ ] File import (picker → copy to storage)

## P2 — Citation

- [ ] Quick Copy (citation to clipboard)
- [ ] Bibliography dialog
- [ ] CSL style selection / loading
- [ ] citeproc engine integration

## P3 — Advanced Zotero

- [ ] Saved search builder UI
- [ ] Duplicate merge UI
- [ ] Related items add/remove UI
- [ ] Group library polish
- [ ] Tag color picker
- [ ] Advanced search pane (condition builder)

## P4 — Localization

- [ ] zh-Hans default
- [ ] en-US fallback
- [ ] String extraction

## P5 — HarmonyOS-specific

- [ ] Cross-device distributed reading
- [ ] Local AI / on-device summarization
- [ ] System share sheet integration
- [ ] Multi-window / freeform support
